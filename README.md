# EV-forecast
Develop EV-charging-demand-forecast-algorithm based on ACN-Data Dataset from Caltech. Data set is avaiable from [ACN-Data](https://ev.caltech.edu/dataset).  

### 当面の目標  
ベイズ推定による確率的予測を実装する
